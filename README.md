# PetHub 🐾
PetHub is a Java-based Pet Care and E-Commerce web application developed to simplify pet care services and online pet product shopping through a single platform.

## Features
* User Registration and Login
* Pet Product Management
* Shopping Cart Functionality
* Wishlist Management
* Veterinary Appointment Booking
* Product Reviews and Ratings
* Search Functionality
* Admin Dashboard and Management
* Responsive User Interface

## Technology Stack
* Java
* JSP
* Servlets
* MySQL
* HTML
* CSS
* JavaScript
* MVC Architecture

## Database
The project uses MySQL as the backend database. The database schema is provided in the `pethub.sql` file.

## Project Structure
* `PetHubProject/` - Main source code of the application
* `pethub.sql` - Database script
* `ER Diagram.jpg` - Database ER Diagram

## How to Run the Project
1. Import the project into Eclipse IDE.
2. Configure Apache Tomcat server.
3. Import the `pethub.sql` file into MySQL.
4. Update database credentials in the project configuration files.
5. Run the application on the server.
